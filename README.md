## Notes

Create .env file with:

``` 
AUTH_TOKEN=your_auth_token
```

Uses Ruby 2.3.1 with rbenv or rvm

```
gem install bundler
g clone repo_url
cd
bundle
/bin/rails db:seed
bundle exec rspec
```
